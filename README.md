# ItalianMealsAPP

# Passi per installare il mio progetto : 

1. Copiare l'indirizzo del mio git :
 git clone https://github.com/danielenigra-creator/ItalianMealsAPP.git

2. Spostarsi nella cartella ItalianMealsApp con questo comando :
cd ItalianMealsApp 

3. Installare il pacchetto di node NPM :
Digitare nel cmd npm install

4. Dopo aver fatto questi passaggi dovrai digitare:   
npx expo start.

5. Puoi simulare l'applicazione su : Smartphone , Emulatore Android e Web
Per usarlo su smartphone dovrai scaricare l'applicazione ExpoGo e dovrai inquadrare 
il QR Code.
Per usare l'emulatore dovrai scaricare Android Studio e creare un dispositivo virtuale, cliccando sui tre puntini
appena avvii il programma e clicchi "new virtual machine".
Installato l'emulatore in npx digitare la lettera "a" per fare partire il tuo programma sul dispositivo.
Per provare la tua applicazione sul Web nel file package.json dovrà esserci quusta seguente riga di codice :    "react-native-web": "^0.21.2".
Come installare npx per web : 
Nel terminare dovrai scrivere : npx expo install react-dom react-native-web.
Dopo di chè quando fai npx expo start per far partire l'applicazione potrai digitare "w" e l'applicazione
parte nel web.

I requisiti che devi avere per un giusto funzionamento sono :
Node.js installato nella tua macchina, un emulatore di android oppure
se testo su dispositivo fisico, l'applicazione Expo Go.

La api usata nel mio progetto è la seguente : 
https://www.themealdb.com/api/json/v1/1/filter.php?a=Italian
